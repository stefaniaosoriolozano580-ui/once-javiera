Explicación de etiquetas semánticas

En mi proyecto utilicé diferentes etiquetas semánticas de HTML5 para organizar correctamente la información y darle significado al contenido.

Estructura general

Usé la etiqueta <html lang="es"> para indicar que mi página está en español, lo cual ayuda a la accesibilidad y a los navegadores.

En el <head> incluí configuraciones importantes como:
- <meta charset="UTF-8"> para que se vean bien los caracteres especiales.
- <meta name="viewport"> para que la página sea adaptable a dispositivos móviles.
- <title> para definir el nombre de la página.

 Estructura del contenido

Dentro del <body> organicé la página en tres partes principales:

- <header>: lo utilicé para colocar mi nombre y una frase personal, ya que es lo primero que se ve en la página.
- <main>: aquí incluí el contenido principal de mi tarjeta personal.
- <footer>: lo usé para colocar mi información de contacto.

 Uso de secciones

Dentro de <main> utilicé dos etiquetas <section> para dividir el contenido:

- Una sección para "Sobre mí", donde describo quién soy.
- Otra sección para "Habilidades y pasatiempos".

Esto permite organizar mejor la información.

Etiquetas de contenido

- <h1>, <h2>, <h3>: las utilicé para los títulos y subtítulos, organizando la información de forma jerárquica.
- <p>: para escribir los párrafos de texto.
- <ul> y <li>: para mostrar una lista de habilidades y pasatiempos.

 Etiquetas semánticas específicas

- <time>: la utilicé para indicar el año 2023, que es cuando empecé a aprender tecnología y HTML.
- <strong>: la usé para resaltar "HTML básico" como una habilidad importante.
- <em>: la utilicé en mi frase personal para darle énfasis.

 Información de contacto

En el <footer> usé la etiqueta <address> para colocar mi correo y ciudad (Medellín, Colombia), ya que esta etiqueta está diseñada específicamente para información de contacto.
