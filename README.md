<h2 style="text-align:center; color:darkblue;">
Cómo creé y guardé mi repositorio en GitHub
</h2>

<h3>Cómo creé mi repositorio</h3>

<ol>
  <li>Primero abrí mi navegador de internet y entré a la página github.com.</li>

  <li>Después inicié sesión con mi cuenta de GitHub. Si alguien no tiene cuenta, primero debe crear una.</li>

  <li>Cuando entré a mi cuenta, busqué el botón que dice "New repository" y hice clic allí.</li>

  <li>Luego escribí el nombre que quería ponerle a mi repositorio.</li>

  <li>Seleccioné la opción "Public" para que el repositorio se pueda ver.</li>

  <li>Después presioné el botón "Create repository" y así se creó mi repositorio.</li>
</ol>

<h3>Cómo guardé mi repositorio</h3>

<ol>
  <li>Una vez dentro del repositorio, hice clic en el botón "Add file".</li>

  <li>Después seleccioné la opción "Create new file".</li>

  <li>Escribí el nombre del archivo: <strong>index.html</strong>.</li>

  <li>Luego copié y pegué mi código HTML dentro del archivo.</li>

  <li>Finalmente bajé hasta el final de la página y presioné el botón "Commit changes" para guardar el repositorio.</li>
</ol>

<p style="text-align:center;">
<strong>Así fue como creé y guardé mi repositorio en GitHub paso a paso.</strong>
</p>
